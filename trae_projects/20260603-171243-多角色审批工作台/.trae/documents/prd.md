## 1. 产品概述

多角色审批工作台是一款面向企业审批管理场景的纯前端单页应用，涵盖申请列表、审批流节点可视化、角色权限管理、操作记录追踪、统计图表和异常提醒六大模块。目标用户为企业审批管理员和各级审批人，解决多角色审批流程中信息割裂、越权操作难以发现、审批流异常缺乏可见反馈等痛点。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 超级管理员 | 系统预设 | 查看所有申请、管理审批流、分配角色、查看统计、处理异常 |
| 部门主管 | 管理员分配 | 审批本部门申请、查看本部门操作记录 |
| 财务审核 | 管理员分配 | 审批财务类申请、查看财务相关记录 |
| 普通员工 | 管理员分配 | 提交申请、查看本人申请和记录 |

### 2.2 功能模块

1. **审批工作台主页面**：申请列表、审批流节点、角色权限面板、操作记录、统计图、异常提醒

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 审批工作台 | 申请列表 | 展示所有审批申请，支持筛选（按状态/类型/申请人），点击行项联动右侧详情面板 |
| 审批工作台 | 审批流节点 | 可视化展示选中申请的审批流，标注当前节点、已通过/拒绝/待审批节点，缺审批人节点红色高亮 |
| 审批工作台 | 角色权限面板 | 展示当前用户角色和权限，执行审批操作时校验权限，越权操作弹出警示反馈 |
| 审批工作台 | 操作记录 | 展示选中申请的所有操作历史，记录为空时显示空状态提示，记录包含操作人、时间、操作类型、备注 |
| 审批工作台 | 统计图 | 饼图展示审批状态分布，柱状图展示各部门审批量趋势 |
| 审批工作台 | 异常提醒 | 实时展示异常项（缺审批人、审批超时、越权尝试），支持点击跳转对应申请 |

## 3. 核心流程

用户进入工作台后，左侧为申请列表，可筛选和搜索。点击某条申请后，中间区域展示该申请的审批流节点和操作记录，右侧展示角色权限面板。顶部横幅显示异常提醒。底部统计图汇总全局审批数据。审批操作触发权限校验：若当前角色无权操作，弹出越权提示；若审批流节点缺少审批人，节点标红并触发异常提醒；若操作记录为空，显示空状态插画。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝灰 (#1E293B) + 琥珀色强调 (#F59E0B)
- 辅助色：翠绿 (通过 #10B981)、玫红 (拒绝 #EF4444)、石板灰 (待审批 #94A3B8)
- 按钮风格：圆角微凸，主操作实色填充，次操作描边
- 字体：DM Sans（正文）+ Space Grotesk（标题/数据）
- 布局风格：三栏式工作台布局，左侧列表、中间详情、右侧面板
- 图标风格：Lucide 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 审批工作台 | 申请列表 | 筛选栏 + 表格列表，行hover高亮，选中行左侧蓝色边条，状态标签彩色 |
| 审批工作台 | 审批流节点 | 横向流程图，节点圆形/菱形，连线动画，当前节点脉冲动画，缺人节点红色虚线 |
| 审批工作台 | 角色权限面板 | 卡片式角色标签，权限列表勾选态，越权操作红色边框闪烁 |
| 审批工作台 | 操作记录 | 垂直时间线，操作类型图标区分，空状态居中插画 + 文案 |
| 审批工作台 | 统计图 | 饼图 + 柱状图并排，hover数据提示，渐变色填充 |
| 审批工作台 | 异常提醒 | 顶部横幅，异常项带闪烁图标，可折叠，点击跳转联动 |

### 4.3 响应式

桌面优先，三栏布局。平板下中间和右侧面板可切换标签页。移动端改为单栏堆叠。
